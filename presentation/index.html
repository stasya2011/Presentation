What are design patterns?

Design patterns are typical solutions to commonly happening problems in software design.

Why should we learn patterns?

  • You can use proven solutions.
  • You seldom can be wrong because use code standardization.
  • Design patterns define a common language that you and your teammates can use to communicate more efficiently.

The Singleton Pattern

Singleton is a creational design pattern that lets you ensure that a class has only one instance, while providing a global access point to this instance.

plus minus
✔ Only a single instance. ✘ Violates the Single Responsibility Principle.
✔ Global access point to that instance. ✘ Сan mask bad design, for instance.
✔ Initialized only when it’s requested for the first time. 
							
								let instance = null;
								function User(name, age) {
									if(instance) {
										return instance;
									}
									instance = this;
									this.name = name;
									this.age = age;
									return instance;
								}
								const user1 = new User('Peter', 25);
								const user2 = new User('Mark', 24);
								console.log(user1 === user2); // true

Factory Method

Factory Method is a creational design pattern that provides an interface for creating objects in a superclass, but allows subclasses to alter the type of objects that will be created.

plus minus
✔ Open/Closed Principle. ✘ The code may become more complicated;
✔ Single Responsibility Principle

We have two classes for create objects 


							class Car{
								constructor(options) {
									this.doors = options.doors || 4;
									this.state = options.state || 'brand new';
									this.color = options.color || 'white';
								}
							}
							class Truck {
								constructor(options) {
									this.doors = options.doors || 4;
									this.state = options.state || 'used';
									this.color = options.color || 'black';
								}
							}

This is the Factory 

						
						class VehicleFactory {
							createVehicle(options) {
								if(options.vehicleType === 'car') {
									return new Car(options);
								}
									else if(options.vehicleType === 'truck') {
										return new Truck(options);
									}
							}
						}

Instances of the Factory was created 

						
							const factory = new VehicleFactory();
							const car = factory.createVehicle({
								vehicleType: 'car',
								doors: 4,
								color: 'red',
								state: 'Brand New'
							});
							const truck= factory.createVehicle({
								vehicleType: 'truck',
								doors: 2,
								color: 'white',
								state: 'used'
							});

Check: 

						
							//Car {doors: 4, state: "Brand New", color: "silver"}
							console.log(car);
							// Truck {doors: 2, state: "used", color: "white"}
							console.log(truck);
							console.log(car instanceof Car); // true
							console.log(truck instanceof Truck); // true
